YouTube Data API v3 Client Library for C++

If you are using a default build environment as documented in the Google APIs Client Library for C++ Installation Guide then:

  1. Move this directory to the service_apis directory in the build root.
  2. Run make from the root directory

To use the specialized service library, consumers may either:

  1. Add this directory to your INCLUDE path.
    • In your CMakeLists.txt file, add the following:
      INCLUDE_DIRECTORIES(${GOOGLEAPIS_SERVICE_REPOSITORY_DIR}/{{ api.module.name })
    • Add the following target_link_libraries for your build target my_build_target:
      target_link_libraries(my_build_target
  2. Either include the single monolithic header file
    #include "/.h"
  3. Or include only the individual header files explicitly needed.

    The provided CMakeLists.txt file is a CMake file compatible with those in the Google APIs Client Library for C++ SDK.

    See Getting Started with the Google APIs Client Library for C++ for more information on how to use these service API libraries.
